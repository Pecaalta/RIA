### Tecnologías necesarias para correr el proyecto
- Gestor de paquetes NPM
- Angular
- Angular CLI

### Comandos para correr el proyecto
1. Posicionarse dentro del directorio del proyecto y ejecutar el comando **npm install**.
1. Ejecutar el comando **ng s -o**, este correra el programa y lo abrira en el navegador.
